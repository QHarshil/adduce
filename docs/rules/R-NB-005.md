# R-NB-005 — No absolute/local paths in notebook cells

**Category:** Notebooks  
**Weight:** 2

## Why it matters

Absolute paths guarantee the notebook fails on any machine but the author's.

## Suppressing

Inline, on the reported line: `# adduce: ignore=R-NB-005`  
Project-wide, in `adduce.toml` or `[tool.adduce]`: `ignore = ["R-NB-005"]`

Suppressed findings still appear in reports, marked as ignored.
