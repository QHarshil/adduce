# R-NB-008 — Result-bearing notebooks have a script equivalent

**Category:** Notebooks  
**Severity:** low  
**Weight:** 2

## Why it matters

Notebooks that produce paper results need a script/jupytext/papermill twin: scripts run headless, diff cleanly, and slot into pipelines; notebooks do none of that.

## Suppressing

Inline, on the reported line: `# adduce: ignore=R-NB-008`  
Project-wide, in `adduce.toml` or `[tool.adduce]`: `ignore = ["R-NB-008"]`

Suppressed findings still appear in reports, marked as ignored.
