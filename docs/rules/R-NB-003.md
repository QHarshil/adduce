# R-NB-003 — Hidden-state risk (gaps in execution counts)

**Category:** Notebooks  
**Severity:** low  
**Weight:** 2

## Why it matters

Gaps mean cells were run and then deleted or edited: the kernel carried state the committed code no longer contains.

## Suppressing

Inline, on the reported line: `# adduce: ignore=R-NB-003`  
Project-wide, in `adduce.toml` or `[tool.adduce]`: `ignore = ["R-NB-003"]`

Suppressed findings still appear in reports, marked as ignored.
