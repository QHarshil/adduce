# R-NB-004 — No !pip install inside notebook cells

**Category:** Notebooks  
**Severity:** low  
**Weight:** 2

## Why it matters

In-cell installs build an environment that exists only in that kernel session and silently diverges from the declared dependencies.

## Suppressing

Inline, on the reported line: `# adduce: ignore=R-NB-004`  
Project-wide, in `adduce.toml` or `[tool.adduce]`: `ignore = ["R-NB-004"]`

Suppressed findings still appear in reports, marked as ignored.
